# Agenda de Prioridades

## Projeto
O projeto "Agenda de Prioridades" é uma ferramenta projetada para ajudar usuários a gerenciar suas tarefas e prioridades de forma eficiente. O objetivo principal é permitir que os usuários organizem suas obrigações, planejem suas atividades e acompanhem seu progresso.

## Funcionalidades
- **Adição de Tarefas**: Os usuários podem adicionar tarefas com prazos.
- **Prioridade**: Atribuição de níveis de prioridade para cada tarefa.
- **Visualização**: Interface para visualizar tarefas pendentes e concluídas.
- **Relatórios**: Geração de relatórios sobre o progresso das tarefas.

## Tecnologias Utilizadas
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js, Express
- **Banco de Dados**: MongoDB

## Como Executar o Projeto
1. Clone o repositório:
   ```bash
   git clone https://github.com/arjunpardiz08-hue/agenda-de-prioridades.git
   ```
2. Navegue até o diretório do projeto:
   ```bash
   cd agenda-de-prioridades
   ```
3. Instale as dependências:
   ```bash
   npm install
   ```
4. Inicie o servidor:
   ```bash
   npm start
   ```
